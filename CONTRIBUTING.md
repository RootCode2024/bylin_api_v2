# Contributing to bylin_api_v2

Merci de votre intérêt pour contribuer à **bylin_api_v2**
Toute contribution est la bienvenue, tant qu’elle respecte les règles ci-dessous.

---

## Prérequis

- PHP >= 8.x
- Composer
- Laravel
- Git

---

## Installation du projet

```bash
git clone https://github.com/TON_USERNAME/bylin_api_v2.git
cd bylin_api_v2
composer install
